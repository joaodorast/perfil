O que é CSS?

CSS, ou Cascading Style Sheets, é uma linguagem utilizada para descrever a apresentação de um documento HTML. Ela controla o layout, cores, fontes e muito mais.

Características do CSS

  • Separação de conteúdo e apresentação.
  • Responsividade e layouts fluidos.
  • Controle detalhado sobre estilos.
