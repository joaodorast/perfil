O que é JavaScript?

JavaScript é uma linguagem de programação que permite implementar funcionalidades complexas em páginas da web. Ela é usada principalmente para fornecer interatividade nas páginas.

Características do JavaScript

  • É uma linguagem de programação de alto nível.
  • Suporte a programação orientada a objetos.
  • Grande comunidade e vasto ecossistema.
