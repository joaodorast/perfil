O que é HTML?

HTML, ou HyperText Markup Language, é a linguagem padrão usada para criar páginas da web. Ela fornece a estrutura básica de um site, que é aprimorada pela CSS e JavaScript.

Características do HTML

  • Estrutura de documentos com tags.
  • Facilidade de uso.
  • Amplamente suportado por navegadores.
